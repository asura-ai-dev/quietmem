## 重要判断

- `create<UiState>()((set) => ({ ... }))` の curried 記法を使用。zustand v4.4+ の TypeScript 推奨形式で、state を type parameter で明示できる。チケットに明示指示あり。
- Action 名はチケット通り `setRoute` / `setActiveTab` / `toggleDrawer` / `setDrawerOpen` / `setDrawerTab`。命名規則を崩さない。
- `toggleDrawer` は `set((state) => ({ drawerOpen: !state.drawerOpen }))` で実装。現在値に依存するため関数形 set を採用。
- Phase 1 では slice 分割しない (チケット Implementation Notes 通り)。
- `projectStore` / `agentStore` 等のドメインキャッシュ store はこのチケットの scope 外 (ui-shell.md にその旨あり)。
