## Key Decisions

- アイコンは placeholder PNG を Python で生成して配置する方式を採用。Tauri v2 の codegen は `bundle.icon = []` でも `icons/icon.png` の存在を要求するため、空配列指定だけでは `cargo check` が通らない。ticket の Implementation Notes にある「minimal に持ってくる」指針に合致する対応。
- `Cargo.lock` は `src-tauri/.gitignore` に含めず、リポジトリにコミットされる方針 (アプリケーション crate のベストプラクティス)。ticket の「含めない設定を検討」という表現は検討の余地を示すもので、アプリケーション (library ではなく) なので lock を保持する判断。
