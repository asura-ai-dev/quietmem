## Frontend-design 出力サマリ

`/frontend-design:frontend-design` を呼び、以下のデザイン方針を確定:

- **コンセプト**: "Editorial caution band" — モーダルではなく雑誌のサイドバー的な inline 警告パネル
- **アクセント**: 上端 3px の amber テープストリップ (`::before` 擬似要素) で「警告帯」を表現。アイコンや派手なグラデーションは使わず、restraint で深刻さを伝える
- **タイポ**: 見出し (sans 600 + slight tracking) + eyebrow (mono uppercase, amber tracking 0.12em) + dt/dd の対比で雑誌的な格式
- **対比表現**: 「引き継がれる項目」を sage (`var(--accent-primary)`)、「引き継がれない項目」を amber (`var(--accent-attention)`) で並置。先頭に 2px×14px の縦バー marker を配置 (アイコンの代替)
- **モーション**: 160ms ease-out + 4px y-offset の入場アニメ (CSS only)
- **トークン制約**: raw 色値ゼロ。tokens.css の alias (`--bg-surface`, `--bg-surface-2`, `--color-amber-300/500/700`, `--color-gray-900`, `--accent-primary`, `--accent-attention`, `--color-danger`, `--border-subtle`, `--font-sans`, `--font-mono`) のみを使用

仕様文言 (`agent-docs/spec.md` §4.5) は固定要件として一字一句変更せず保持。

## 実装した 2 ファイル概要

### 1. `AgentDuplicateConfirm.tsx`

- React import: `useEffect`, `useId` のみ (チケット指定通り)
- props: `agent: Agent`, `loading?: boolean`, `errorMessage?: string | null`, `onCancel: () => void`, `onConfirm: () => void`
- `useEffect` 内で `window.addEventListener('keydown', ...)` → ESC キーで `onCancel`、unmount 時に cleanup
- `useId()` で `headingId` / `descId` を生成 (Phase 1 既存パターン)
- `<aside role="alertdialog" aria-modal="false" aria-labelledby={headingId} aria-describedby={descId}>`
- 構造: header (eyebrow + h4 title) / body (intro + dl 対比表 + error) / footer (cancel + confirm)
- loading 中はボタン disabled + confirm が "複製中…" 表示
- default export
- spec.md §4.5 文言 ("Agent 固有の memory (raw / curated)、active worktree、status (idle で開始)") を完全一致で記載
- 引き継がれる項目テキストは「name (末尾に "(copy)")、role、adapter type、prompt path、config path」(チケット snippet と等価。読点区切り、`&quot;` は JSX の lint 回避目的)

### 2. `AgentDuplicateConfirm.module.css`

- `.root`: margin-top var(--space-3) / 1px var(--color-amber-500) border / var(--bg-surface) bg / var(--radius-md) / flex column gap var(--space-3) / position relative + overflow hidden / 入場アニメ
- `.root::before`: 上端 3px var(--color-amber-500) accent strip (テープストリップ表現)
- `@keyframes agentDuplicateConfirmEnter`: opacity 0→1 + translateY 4px→0
- `.header` / `.eyebrow` (mono uppercase amber tracking) / `.title` (sans 600)
- `.body` / `.intro` / `.agentName` (mono 600)
- `.list`: `display: grid` / `grid-template-columns: max-content 1fr` / 上下 border-subtle で囲って register 感
- `.copy` (var(--accent-primary)) / `.skip` (var(--accent-attention))
- `.markerCopy` / `.markerSkip`: 2px×14px の縦バー (sage / amber)
- `.error`: var(--color-danger) text + var(--bg-surface-2) bg + 2px left border
- `.footer`: flex end gap
- `.cancel`: var(--bg-surface-2) bg + var(--fg-primary) fg + hover で var(--bg-app) + focus-visible outline
- `.confirm`: var(--color-amber-500) bg + var(--color-gray-900) fg + var(--color-amber-700) border + 600 weight + hover で amber-300 + focus-visible amber-300 outline
- `.cancel:disabled, .confirm:disabled`: opacity 0.6 + cursor not-allowed
