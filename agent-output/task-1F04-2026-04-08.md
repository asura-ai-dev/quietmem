## Notes

- `AgentEditForm` は `key={selectedAgent.id}` でマウントされるため、別 agent を選択し直した場合は完全に再マウントされる。`useEffect` による state 再同期はその保険として残しつつ、同一 agent が store 上で更新されたケース (refresh で props が差し替わる) にも対応する
- Worktree が 0 件の場合の disabled + hint は ticket 要件通り。disabled 時も空文字 `""` を value として持つため submit 時の型は一貫 (`activeWorktreeId: null`)
- select の chevron は raw SVG やアイコンライブラリを使わず CSS linear-gradient 2 本で描画 (`ui-shell.md` のアイコン非導入方針を踏襲)
- agentStore.updateAgent は成功後に返却 agent の `projectId` で `refreshAgents` を呼ぶ既存実装のため、form から `refreshAgents` を明示呼び出ししなくても「保存後 refreshAgents」要件は満たされる
